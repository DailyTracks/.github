### 프로젝트 개요
DailyTracks는 지도 위에 사람들의 일상을 기록하는 라이프 로깅 플랫폼입니다. 사용자는 지도를 클릭하여 해당 지역의 일상 기록을 확인할 수 있습니다. 지도는 실시간으로 해당 지역의 인기도에 따라 색상이 변경됩니다.
